# Opportunities

Gap-shaped ideas surfaced during the voice note.

1. **Device-agnostic agentic workspace.** A real product-shaped gap: many small utilities, one home, web + mobile, MCP-aware, skill-aware.
2. **Latent value extraction.** Retroactive pipelines: chat history → memory, prompt history → prompt library, output history → wiki / knowledge base.
3. **Skills as the portability layer.** Instead of syncing MCPs across devices, sync skills and point them at stable remote MCPs (home server / VPS + Tunnel / Tailscale).
4. **Proactive memory agent.** One that detects gaps ("your job context looks stale — want to update?") and drives interview-style ingestion.
5. **Informal eval harness as a product.** Rotate models through a recurring task (e.g., podcast script). Capture quality deltas cheaply.
6. **Multi-perspective / LLM-council tooling for civic issues** — brainstorming, stakeholder role-play, entrenched-problem ideation (beyond marketing proxies à la Rally).
7. **OSINT → personal alert pipeline.** Telegram keyword ingestion → geo-scoped actionable alerts (shelter / no-shelter). Civilian-grade preparedness tooling.
8. **MCP file-transport standard.** Base64 / upload-aware MCP pattern + aggregator that understands remote hosts. A worthwhile open-source contribution.
9. **Podcast-as-personal-info-channel** as a repeatable pattern. Cheap enough (~$0.30/ep) that it generalises beyond one user.
10. **Repo-template library** (already in motion). Codify the "workspace repo" pattern so others can adopt it without reinventing folder structure + skills + MCP wiring.
11. **Children's content via image-to-video + voice-cloned characters.** Latent creative project using the existing podcast stack (Chatterbox + stuffed-animal voices + Sora-style video).
12. **Backfill / secondary-route layer.** An extraction service any chat-storing app can feed into; produces memory, prompts, wiki entries downstream.
